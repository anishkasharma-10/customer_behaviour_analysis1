# customer_behaviour_analysis1
# 📊 Data Analytics Project

## 🚀 Overview

This project demonstrates the complete data analytics workflow — from data loading and cleaning to visualization and reporting. The goal is to extract meaningful insights from the dataset and present them through interactive dashboards and reports.

---

## 📂 Dataset

The dataset contains information related to customer purchases, including:

* Customer ID
* Age
* Gender
* Product Category
* Purchase Amount
* Location
* Season
* Payment Method, etc.

---

## 🛠️ Tools & Technologies Used

* **Python** (Pandas, Matplotlib, Seaborn)
* **SQL** (PostgreSQL / MySQL / SQL Server)
* **Power BI** (Dashboard creation)
* **Gamma / PowerPoint** (Presentation)

---

## 🔄 Project Workflow

### 1️⃣ Data Loading

* Loaded dataset using Pandas
* Explored structure using `.head()`, `.info()`, `.describe()`

### 2️⃣ Exploratory Data Analysis (EDA)

* Identified patterns, trends, and outliers
* Visualized data using graphs and charts

### 3️⃣ Data Cleaning

* Handled missing values
* Removed duplicate records
* Standardized column names

### 4️⃣ SQL Analysis

* Performed queries to extract insights
* Used GROUP BY, SUM, AVG, COUNT functions

### 5️⃣ Power BI Dashboard

* Created interactive dashboards
* Used charts like bar, pie, and line graphs
* Added filters for better user interaction

### 6️⃣ Reporting & Presentation

* Summarized key findings in a report
* Created presentation slides using Gamma

---

## 📊 Key Insights

* Identified top-performing product categories
* Analyzed customer purchasing behavior
* Compared spending patterns across gender and age groups
* Observed seasonal trends in sales

---

## 📈 Results

* Cleaned and structured dataset ready for analysis
* SQL queries providing business insights
* Interactive Power BI dashboard
* Final report and presentation
